HOW TO UPDATE MONTHLY
================================================================================

STEP 1: Open "Dados" sheet
STEP 2: Enter actual numbers for the current month
        - Example: If March data, go to column D (Mar)
        - Enter into rows 2-21 for all metrics

STEP 3: All other sheets automatically recalculate:
        - "Painel Executivo" KPI cards update instantly
        - "Marketing" channel metrics recalculate
        - YTD and % Meta columns auto-update
        - All formulas refresh

STEP 4: Review dashboard:
        - Open "Painel Executivo" for executive summary
        - Check "Marketing" for channel performance
        - Review "Projetos" for delivery status

DESIGN PRINCIPLES USED
================================================================================

This dashboard follows Cole Nussbaumer Knaflic's "Storytelling with Data":

1. ELIMINATE CLUTTER
   - No gridlines - cleaner visual
   - Minimal borders - focus on content
   - Generous white space - breathing room

2. STRATEGIC COLOR USE
   - Gray for context
   - Lime for what matters most
   - Orange for alerts
   - Dark backgrounds for emphasis

3. PREATTENTIVE ATTRIBUTES
   - Bold, large numbers grab attention first
   - Size hierarchy shows importance
   - Color directs focus to key metrics

4. INSIGHT-FIRST TITLES
   - Not just metric names
   - Titles that tell the story
   - Examples:
     * "RECEITA vs META — Estamos no caminho certo?"
     * "QUAIS CANAIS TRAZEM MAIOR RETORNO?"
     * "EVOLUÇÃO DO FUNIL DE CONVERSÃO"

5. DATA INTEGRITY
   - Single source of truth (Dados sheet)
   - All calculations via formulas
   - Automatic updates across sheets
   - No manual copying
