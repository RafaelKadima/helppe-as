## Design Philosophy

This dashboard is built on **Storytelling with Data** principles (Cole Nussbaumer Knaflic):

### 1. Eliminate Clutter
- No gridlines (cleaner visual)
- Minimal borders
- Generous white space
- Clean Arial typography

### 2. Strategic Color Use
- **Lime (#CAFF00):** Key metrics, completed work, what matters
- **Gray (#6B6B6B):** Context and labels
- **Orange (#FF6B00):** Alerts and delays
- **Dark Charcoal (#1A1A1A):** Emphasis and hierarchy
- **Steel Blue (#2D4659):** In-progress work

### 3. Preattentive Attributes
- Bold, large numbers grab attention first
- Color directs focus
- Size indicates importance
- Position shows hierarchy

### 4. Insight-First Titles
Not just metrics—titles that explain the story:
- "RECEITA vs META — Estamos no caminho certo?" (Are we on track?)
- "QUAIS CANAIS TRAZEM MAIOR RETORNO?" (Which channels drive return?)
- "EVOLUÇÃO DO FUNIL DE CONVERSÃO" (Funnel evolution)

### 5. Single Source of Truth
- All calculations via formulas
- No manual copying
- Automatic updates across sheets
- Complete transparency

## Technical Specifications

### File Properties
- **Format:** Microsoft Excel 2007+ (.xlsx)
- **Size:** 25 KB (optimized)
- **Sheets:** 6
- **Formulas:** 121 (all validated, 0 errors)
- **Compatibility:** Excel, Google Sheets, Apple Numbers

### Sheet Setup
- All sheets: Landscape orientation, no gridlines
- Freeze panes on row 1 or row 2 (headers always visible)
- Print setup: Fit to 1 page wide

### Fonts
- Primary: Arial 10/11
- Headings: Arial Bold 11/20
- Values: Arial Bold 24 (lime color)

### Number Formats
- Currency: `R$ #,##0` (Brazilian Real, no cents)
- Percentages: `0.0%` (one decimal place)
- Whole numbers: `#,##0`
- Dates: `dd/mm/yy`

### Formulas Include
- Ticket Médio (Revenue ÷ Orders)
- Conversion Rate (Orders ÷ Visitors)
- LTV/CAC Ratio (LTV ÷ CAC)
- YTD accumulations (SUM/AVERAGE)
- % of target progress
- ROAS by channel (Revenue ÷ Investment)

## Support & Maintenance

### Monthly Workflow
1. **First day of month:** Review previous month targets
2. **During month:** Track progress in separate data
3. **End of month:** Enter actual data in Dados sheet
4. **First of next month:** Generate executive briefing

### Annual Maintenance
1. Review and adjust target (Meta) numbers
2. Add new metrics if business focus changes
3. Archive previous year data
4. Reset sheets for new fiscal year

### Troubleshooting
- **Formulas not calculating:** Press Ctrl+Shift+F9 (Force recalc)
- **Colors look different:** Check Excel color settings
- **Data disappeared:** Check if cells accidentally cleared
- **Formulas show as text:** Format cells as Number
