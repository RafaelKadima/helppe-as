## DICAS DE USO

**Para Midjourney:**
- Adicione `--ar 1:1` para quadrado, `--ar 16:9` para horizontal, `--ar 9:16` para vertical
- Adicione `--v 6.1` para a versão mais recente
- Adicione `--s 250` para mais estilo ou `--s 50` para mais fidelidade ao prompt
- Use `--no text, letters, words` se a ferramenta estiver gerando texto ilegível

**Para DALL-E / ChatGPT:**
- Use os prompts diretamente
- Peça ajustes iterativos: "agora mude a cor do fundo para branco"

**Para Leonardo AI:**
- Use o modelo "PhotoReal" para mockups mais realistas
- Ative "Prompt Magic" para melhor aderência às instruções

**Cores HEX para referência rápida:**
- `#0A0A0A` — Volt Black (fundo/dominante)
- `#CAFF00` — Electric Lime (destaque/logo/CTA)
- `#2D4659` — Steel Blue (alternativa sóbria)
- `#FF6B00` — Warning Orange (promoções)
- `#F5F5F3` — Off White (fundos claros)
- `#1A1A1A` — Dark Charcoal (containers)
- `#6B6B6B` — Concrete Gray (textos secundários)
